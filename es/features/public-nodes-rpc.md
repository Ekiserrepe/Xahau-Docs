# Nodos Públicos (RPC)

* Websocket
  * `wss://xahau.network`
  * `wss://xahau.org` (alias, algunos bloqueadores de anuncios bloquean .network)
* HTTP POST RPC
  * `https://xahau.network`
  * `https://xahau.org` (alias, algunos bloqueadores de anuncios bloquean .network)
* Definiciones de red (Codec binario, ...)
